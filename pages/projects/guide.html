TOOL GUIDE

Everything here runs 100% in your browser — nothing is uploaded, no account, no server. Each tool exports files you keep. This page explains all of them end to end.

stream.mk (OBS widgets) chat bridge setup window theme bootanim.mk schedule.mk emote.mk

stream.mk — OBS browser-source widgets

A generator that builds self-contained .html overlays for OBS (or Streamlabs). Pick a widget, style it, hit export, and add the file to OBS — nothing phones home after download.

Add any widget to OBS: Sources → + → Browser → Local file → pick the exported .html → set the width/height to your canvas (usually 1920×1080).

open stream.mk →

Chat merger

Merges live chat across platforms, priority TikTok › Twitch › Facebook › YouTube.

  • Twitch — works with zero setup: just type your channel name.
  • YouTube — needs your free Data API key + the live video id.
  • TikTok & Facebook — come through the local bridge (see below).

Scene screens

Start / BRB / End (centered hero, per-flavor position & text-align), a chat layout (resizable, swappable cam + chat split), and a game layout (3 freely-placed elements — game window, cam, chat — each with editable position & size). Custom background image, removable pixelated glow, and a download-as-PNG button (names files start_1.png, game_1.png…).

A PNG is a still image — it won't run the countdown. For a live countdown or clock, drop the Timer / clock widget on top of the scene in OBS.

Timer / clock

A live clock, a count-up stream timer, or a countdown from HH:MM:SS.

Alert box

Full StreamElements/TikFinity-style alerts. Add one row per event (follow, sub, gift, raid, tip…), each with its own title + subtitle, image/GIF, sound, color, duration, and minimum amount (for tiered variations). Message tokens: {user} {amount} {tier} {months} {message} {count}. Add ?test to the file URL to demo every type.

Social banner

A rotating lower-third — the bar stays put, the text swaps (one message per line).

Stinger transition

A full-screen transition that covers the cut and flashes your glitch wordmark. 6 styles: wipe (4 directions), bars, shutter (blinds), split, iris (circle), zoom. Plays once on load; the preview loops.

OBS's built-in Stinger transition needs a video file, so the tool has an ⬇ export stinger.webm (alpha) button that encodes the animation frame-by-frame to an alpha .webm in your browser (WebCodecs — fast, not a screen-recording). In OBS: Settings → Scene Transitions → + → Stinger, pick the file, and set the Transition Point to the value the tool shows. (Alpha works in Firefox & recent Chrome; if a browser drops the alpha you'll get a black background — use Firefox instead.) You can also use the exported stinger.html as a plain triggered browser-source overlay.

Window theme (global)

Every widget has a Style setting: simple or window. In window mode each element turns into its own little window/notification — chat messages become notification cards with a username title bar, the scene's cam/chat/game boxes each get a title bar, the timer and alert become notification windows, and the social banner becomes a desktop taskbar (start button + clock). Editable title-bar color, Windows/macOS buttons, and border color/width.

Fonts

Each widget's Font control has three sources: this site's fonts (embedded into the file, works offline in OBS), Google Fonts (searchable — loads over the network), or upload your own .ttf/.otf/.woff (embedded).

Chat / alert bridge (TikTok & Facebook)

Browsers can't read TikTok/Facebook live events directly, so a tiny local Node bridge does it and feeds your overlay over a WebSocket. One-time setup:

  1. Install Node.js (nodejs.org).
  2. In stream.mk's Chat tab, download bridge.js and the launcher for your OS: run-bridge.bat (Windows) or run-bridge.sh (Linux/macOS). Keep all of them in one folder.
  3. Windows: double-click run-bridge.bat. Linux/macOS: chmod +x run-bridge.sh then ./run-bridge.sh. The script installs the dependencies the first time and asks for your TikTok username.
  4. Leave it running while you stream. In the widget, set the Bridge field to ws://localhost:8787 (the default).
Facebook needs your own Page access token + live video id via the Graph API — see the comments inside bridge.js for where to plug it in.

bootanim.mk — Android boot animation maker

Builds an Android bootanimation.zip in the browser. Two modes:

  • Text mode — type a fake boot log (lines starting with [OK] [FAILED] [WARN] [INFO] get colored), pick fonts/colors/resolution, optional blinking cursor.
  • Video mode — turn a video into a looping boot animation.

Export gives you a bootanimation.zip (store-compressed, as Android requires). Push it to /system/media/bootanimation.zip (root) or ship it in a Magisk module.

open bootanim.mk →

schedule.mk — weekly schedule maker

Turn your week into one shareable image. Toggle days on/off, set times and activities, pick the theme colors and font, optionally add a background image, then export a PNG in square (1080), wide (1920×1080), or story (1080×1920).

open schedule.mk →

emote.mk — emote / sticker / badge maker

Import art (drag-drop or click), then:

  • Frame — zoom, move, pixelate, optional circle mask.
  • Background removal — key out a color (with a "pick from corners" auto button) + tolerance.
  • Sticker outline — add a colored contour of any thickness.
  • Fill — transparent, or a solid background color.

Export a whole preset at once — Twitch emote (112/56/28), Twitch sub badge (72/36/18), Discord emoji/sticker, YouTube, or generic — in PNG, WebP, or JPG. Multiple sizes come down as a .zip; a single size downloads directly.
